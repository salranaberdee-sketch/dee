# ระบบประเมินผลนักกีฬา (Athlete Performance Evaluation)

## ภาพรวม
ระบบสำหรับติดตามและประเมินผลนักกีฬาแต่ละคน โดยวัดจากการเข้าร่วมกิจกรรม การฝึกซ้อม และผลงานโดยรวม เพื่อคัดแยกนักกีฬาที่มีผลงานดีและต้องปรับปรุง

## Acceptance Criteria

### AC1: บันทึกการเข้าร่วม (Attendance Tracking)
- ระบบต้องบันทึกสถานะการเข้าร่วมของนักกีฬาในแต่ละกิจกรรม
- สถานะประกอบด้วย: มาตรงเวลา (on_time), มาสาย (late), ลา (leave), ขาด (absent)
- Coach/Admin สามารถบันทึกการลาล่วงหน้าได้
- ระบบคำนวณอัตราการเข้าร่วมอัตโนมัติ

### AC2: สถิติการฝึกซ้อม (Training Statistics)
- แสดงจำนวนครั้งที่ฝึกซ้อมในช่วงเวลาที่กำหนด (สัปดาห์/เดือน)
- แสดงชั่วโมงฝึกซ้อมรวม
- แสดงคะแนนเฉลี่ยจาก Coach (rating 1-5)
- เปรียบเทียบกับเป้าหมายที่ตั้งไว้

### AC3: ระดับผลงาน (Performance Tier)
- ระบบจัดระดับนักกีฬาอัตโนมัติตามเกณฑ์:
  - ⭐ ดีเยี่ยม (Excellent): เข้าร่วม ≥90%, rating ≥4.5
  - ✓ ดี (Good): เข้าร่วม ≥75%, rating ≥3.5
  - ○ ปานกลาง (Average): เข้าร่วม ≥50%, rating ≥2.5
  - ⚠ ต้องปรับปรุง (Needs Improvement): ต่ำกว่าเกณฑ์

### AC4: Dashboard สรุปผล
- Admin/Coach เห็นภาพรวมนักกีฬาทั้งหมดในชมรม
- กรองตามระดับผลงาน
- จัดอันดับนักกีฬาตามคะแนนรวม
- Export รายงานได้

### AC5: รายงานรายบุคคล
- นักกีฬาเห็นสถิติของตัวเอง
- แสดงกราฟแนวโน้มการพัฒนา
- แสดงจุดที่ต้องปรับปรุง
